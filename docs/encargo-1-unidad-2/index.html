Encargo 1 Unidad 2

Moldes Fusion 360

Para este encargo se diseñaron 3 moldes mediante el uso de el software Fusion 360. El primero de estos fue un molde de corte circular, el segundo un molde de vaciado y el tercero un molde tipo prensa que permite tener una geometria especifica tanto en la parte inferior como en la superior del elemento que se desee formar. En todos los moldes se tuvo en cuenta un ángulo de desmoldado para permitir una mejor extracción de lo que se desee crear.
